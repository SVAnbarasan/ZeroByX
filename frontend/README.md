# Frontend

This directory contains the React (TypeScript) frontend for the project.

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm run dev
   ```

## Project Files
- `src/` - Source code
- `public/` - Static assets
- `index.html` - Main HTML file

## Environment Variables
- Place any secrets or config in a `.env` file (not committed)
