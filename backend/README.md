# Backend

This directory contains the Python backend code for the project.

## Setup

1. (Optional) Create a virtual environment:
   ```sh
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
2. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```

## Running the Server

To start the backend server:
```sh
python server.py
```

## Project Files
- `server.py` - Main backend server
- Other `.py` files - Agents and utilities

## Environment Variables
- Place any secrets or config in a `.env` file (not committed)
